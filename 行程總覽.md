## 每日行程主題

### 第 1 天｜6/20（六）
## 「啟程——飛越海峽，踏入古都」

從桃園出發，越過海峽，踏上關西的土地。在機場完成入境手續，搭乘 HARUKA 特急穿越大阪灣地景直抵京都，在夜色中抵達五條飯店，開始七天七夜的旅程。

| 時間 | 地點 |
|------|------|
| 12:15 | 臺灣桃園國際機場 出發 |
| 19:00 | 關西國際機場 抵達、辦理入境 |
| 21:00 | 搭乘 JR HARUKA → 京都站 → 五條 |
| 22:38 | 維瑟爾坎帕納京都五條飯店 入住 |

---

### 第 2 天｜6/21（日）
## 「茶韻鳥居——宇治茶鄉與千本鳥居」

前往宇治，參觀世界遺產平等院，品嘗中村藤吉百年抹茶料理，漫步宇治橋通採購茶葉伴手禮。下午轉往伏見稻荷大社，穿越萬本鳥居走入山林信仰。傍晚在高木批發超市大肆採購，夜晚回到京都車站覓食。

| 時間 | 地點 |
|------|------|
| 09:38 | 平等院（世界遺產） |
| 11:48 | 中村藤吉本店 宇治本店（午餐） |
| 13:19 | 宇治橋通商店街振興組合 |
| 14:50 | 伏見稻荷大社 |
| 17:13 | 高木批發超市三条店 |
| 19:55 | 京都車站周邊晚餐 |

---

### 第 3 天｜6/22（一）
## 「東山古道——石板坡道與河原町之夜」

從北邊的下鴨神社古森出發，穿越城市前往東山；清水寺、産寧坂、二寧坂、一年坂、石塀小路一路漫步，感受京都最完整的町家街區風貌。傍晚走入八坂神社與花見小路，在鴨川納涼床感受京都夏日，夜晚在四条河原町、新京極與寺町京極盡情血拚。

| 時間 | 地點 |

| 11:27 | 清水寺 |
| 12:34 | 産寧坂 → 二寧坂 → 一年坂 → 石塀小路 |
| 14:52 | 京都八坂神社 |
| 16:00 | 花見小路通 |
| 16:41 | 鴨川納涼床 |
| 16:55 | iroha 北店 |

### 第 4 天｜6/23（二）
## 「京都最後半日，揮別古都入大阪」

最後一個京都上午，輕鬆走訪京都水族館，感受生態教育；中午前往錦市場品嘗「京都的廚房」小食，逛錦天滿宮與祇園白川水岸。傍晚在人氣名店挽肉と米享用晚餐，搭車移往大阪，夜遊美國村，感受截然不同的都市活力。

| 17:00 | 挽肉と米（京都）晚餐 |
| 18:00 | 搭車移至大阪心齋橋 |
| 19:40 | 美國村夜遊 |

### 第 5 天｜6/24（三）
## 「魔法樂園——環球影城全日狂歡」

一整天獻給大阪環球影城。超級任天堂世界、哈利波特魔法世界、小小兵天地全力衝刺，體驗日本最頂尖的主題樂園。夜晚在美國村的海鮮屋台犒賞全天消耗的體力。

| 時間 | 地點 |
|------|------|
| 07:00 | 日本環球影城（全天） |
| 21:26 | 海鮮屋台おくまん アメ村店（晚餐） |

> **注意**：超級任天堂世界須事先搶整理券（免費）或購買 Express Pass；強烈建議 07:00 前入園搶先排隊。

## 永續里山旅行提醒

- 全程優先使用鐵路與步行，計程車為最後備案
- 自備水瓶、購物袋、手帕，減少一次性耗材
- 在寺社、町家街區與住宅區保持安靜，尊重在地生活
- 採購以耐用、有地方故事的物品為優先，避免過度包裝伴手禮
- 宇治茶文化、鴨川、錦市場、天神橋筋商店街等，皆是感受人與自然、人與地方共生的里山體驗點
